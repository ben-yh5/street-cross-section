Cool little visualization for OSM data in Seattle. Works on all roads in all places, but only Seattle is cached for quick access.

- Click on roads to see their cross-section, according to OSM. Note that there is currently no support for separated infrastructure (like cycletracks).
- Query roads by certain OSM properties. Try "lanes=6 and cycleway=shared\_lane" for terrible places.
- You can import a GPX of a route, and it will show you the cross-section of roads along the route.

Seattle OSM data cached July 2026.
